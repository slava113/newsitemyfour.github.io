HTML

—Hypertext Markup Language”) is the most basic building block of the Web. It defines the content and structure of web content. Technologies other than HTML are commonly used to describe appearance / presentation (CSS) or functionality / behavior (JavaScript) .Hypertext refers to links that connect web pages to each other, either within the same website or between websites. Links are a fundamental aspect of the web. By uploading content to the Internet and linking it to pages created by other people, you become an active member of the World Wide Web. HTML uses markup to display text, images, and other content in a web browser. HTML markup includes special "elements" such as head, title, body, header,footer, article> section, p, div, span, img, aside, audio, canvas, datalist, details, embed, nav, output, progress, video and many others. An HTML element is extracted from other text in the document using "tags", which consist of the element name surrounded by "<" and ">". The element name inside the tag is not case sensitive. That is, it can be written in upper or lower case, or mixed. For example, the title tag can be written as Title, TITLE, or in any other way

CSS

A style or CSS (Cascading Style Sheets, cascading style sheets) is a set of formatting options that is applied to elements of a document to change their appearance. The ability to work with styles has long been included in developed publishing systems and text editors, thereby allowing you to give the text a predefined, predefined look with the click of a button. Now it’s also available to the creators of the site, when the color, text size and other parameters stored in a specific place and easily "screwed" to any tag. Another advantage of styles is that they offer a lot more formatting options than regular HTML CSS is a powerful system that extends the design and layout of web pages.
